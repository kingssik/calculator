javascript로 계산기 만들기

@계산기 기능 정의 및 처리 순서@
 - 1. 숫자 버튼 클릭
 - 2. 클릭 이벤트 발생
 - 3. 클릭 이벤트가 발생한 버튼의 값 저장
 - 4. '=' 버튼 클릭
 - 5. 저장된 값 연산
 - 6. input에 연산된 결과값 출력

@계산기 기본 구조@
html, js 파일 분리
css 파일 생성하지 않음(table + button로 기본골격 형성)

html 파일에 코드를 작성해 기본 구조 형성
- 필요한 요소 : 숫자 입력 공간 / 숫자버튼(0~9) / 연산자 버튼
- 사용한 태그 : table / button / input

javascript
- 변수를 만들어 필요한 값 저장
- 변수를 선언한 다음 버튼의 값을 변수에 저장해야하는데 
  버튼의 값을 저장하기 위해서는 val() 과 $()을 사용
($()은 jQuery의 method, 선택자를 이용해 원하는 HTML 요소를 가져올 수 있게)
- 선택자를 가져올때 id와 class는 #과 . 을 사용
- val()은 HTML 요소에 할당된 value를 가져올 수 있게
